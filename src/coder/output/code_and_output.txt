Here is the complete content of the Python program and its output for calculating the first 10,000 terms of the series \(1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + ...\), which approximates \(\pi\):

```python
# Python code to calculate the first 10,000 terms of the series
def calculate_pi(num_terms):
    total_sum = 0
    for n in range(num_terms):
        term = (-1) ** n / (2 * n + 1)
        total_sum += term
    return total_sum * 4

# Calculate pi using the first 10,000 terms
pi_approximation = calculate_pi(10000)
print(pi_approximation)
```
